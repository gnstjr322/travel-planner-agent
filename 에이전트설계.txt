#참고 https://rudaks.tistory.com/entry/langgraph-Multi-agent-supervisor
langgraph 이용한 multi agent supervisor 설계

supervisor
├── search 에이전트 : 여행지 검색 및 정보 수집
│   ├── 웹 서치 api (툴1)
│   └── map 서치 api(툴2)
│    
├── planner 에이전트 : 여행 일정 계획 수립
│   └── 여행 계획 세워두는 도구(툴3)
│
├── calender 에이전트 : 캘린더 연동 및 일정 관리
│   ├── 캘린더 신규등록 api (툴4)
│   ├── 캘린더 조회 api (툴5)
│   ├── 캘린더 수정 api (툴6)
│   └── 캘린더 삭제 api (툴7)
│
└── share 에이전트: 계획서 공유 기능
    └── 외부 등록 api (툴8)


#참고사항
- 에이전트 생성시 create_react_agent 이용
- 먼저 상위 2개 에이전트에 대해서만 설계하고 실행해보기
- 각 에이전트에서 다음 에이전트로 넘어갈때 사용자한테 할까요? 질문받고 실행
